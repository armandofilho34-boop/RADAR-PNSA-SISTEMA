04

Kanban Board

Gestão visual e fluxo completo das demandas

📋
5 Colunas de Status

O Kanban é dividido em 5 colunas que representam o ciclo de vida de cada demanda:

🟡 A fazer → 🔵 Fazendo → 🟣 Para aprovação → 🔴 Alteração → 🟢 Aprovado

🟡
A Fazer (Aguardando)

Quando uma demanda é criada, ela aparece aqui. O executor designado deve arrastá-la para "Fazendo" quando iniciar o trabalho.

🔵
Fazendo (Em Execução)

A demanda está sendo trabalhada. Ao concluir, o executor arrasta para "Para aprovação". Atenção: não é possível voltar para "A fazer" após iniciar.

🟣
Para Aprovação

A demanda fica travada nesta coluna — apenas o solicitante original ou coordenador pode aprovar ou solicitar alteração. O executor não consegue mais movê-la.

🔴
Alteração Solicitada

Se o revisor solicitar ajustes, a demanda volta para o executor com um comentário do que precisa ser corrigido. Após ajustar, arraste novamente para "Para aprovação".

🟢
Aprovado (Concluído)

O trabalho foi aceito pelo solicitante. A demanda é finalizada e contabilizada nas métricas do dashboard.

🔗
Pipeline Multi-Departamento

Se a demanda envolve mais de um departamento (ex: Design → Vídeo → Social Media), cada etapa segue o mesmo fluxo. Ao ser aprovada em uma etapa, a demanda avança automaticamente para o próximo departamento.

🖱️
Arrastar e Soltar

Arraste os cards entre colunas para atualizar o status. A zona de drop fica destacada em roxo. Cards com cadeado 🔒 estão travados para o seu perfil.

🔍
Filtros e Busca

Use a barra de pesquisa para buscar por nome. Filtre por prioridade (Crítico, Alta, Média, Baixa) ou departamento para encontrar demandas específicas.

🎨
Cores de Prioridade

Cada card tem uma barra lateral colorida: 🔴 Crítico/Urgente, 🟠 Alta, 🔵 Média/Normal, ⚪ Baixa. Isso ajuda a identificar rapidamente as demandas mais urgentes.

05

Fluxo de Aprovação

Como revisar e aprovar demandas

👁️
Para Revisar

Quando o executor finaliza, a demanda aparece em "Para Revisar". O solicitante (ou coordenador) deve revisar.

✅
Aprovar

Se o trabalho estiver correto, clique em "Aprovar". A demanda é marcada como concluída.

✏️
Solicitar Alteração

Se precisar de ajustes, solicite alteração com um comentário. O executor receberá a notificação.
